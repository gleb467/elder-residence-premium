CSS mask для плавной прозрачности:

css.hero-image {
  background-image: url('../images/hero-desktop.webp');
  background-position: left center;
  background-size: cover;
  mask: linear-gradient(90deg, transparent 0%, black 30%, black 100%);
  -webkit-mask: linear-gradient(90deg, transparent 0%, black 30%, black 100%);
}

Псевдоэлементы требуют обязательных свойств для отображения
mask дает настоящую прозрачность (не белое размытие)
Разделение на части лучше для адаптивности
